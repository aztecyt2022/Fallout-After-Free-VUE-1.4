How to change the ps4 vue theme.

You can use what ever vue theme you want for this.

----------------------------------------------------------------------------------------------------------------
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
----------------------------------------------------------------------------------------------------------------
STEP.1
To get started your ps4 has to be jailbroken already and the vue app has to be open
----------------------------------------------------------------------------------------------------------------
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
----------------------------------------------------------------------------------------------------------------
STEP.2
download or open an ftp client of your choice.
+
connect to your ps4 system. when connected go to
+
path #1
mnt/sandbox/CUSA00960_000/download0/themes/your file here
you may have to rename to get the theme to work correctly and remove other folders there(Keep a backup)
+
path #2
You may also find the themes in mnt/sandbox/download/cusa00960/themes/your file here
(Not always if you have both then ignore this 1 and only use the path #1)
----------------------------------------------------------------------------------------------------------------
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
----------------------------------------------------------------------------------------------------------------
Step.3
after you have added the theme folder you can now reset the vue app and reload it.
+
this will update the theme.
----------------------------------------------------------------------------------------------------------------
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
----------------------------------------------------------------------------------------------------------------
possible issues
Audio file may not transfer over if this happens use the download0 folder to get the sfx audio file and replace the 1 in the download0 folder on the ps4.
Themes do not support sfx audio changing at this time.



The vue app must be running to do this and you need an active ftp connection.
